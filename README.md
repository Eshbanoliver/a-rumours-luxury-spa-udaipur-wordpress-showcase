## Technical Stack
- WordPress v6.7+
- PHP 8.2+
- Elementor Pro (or similar builder) + custom CSS for elegant styling, icons, and spa aesthetics
- JavaScript for sliders (testimonials/gallery), smooth interactions
- Plugins: Elementor, contact/booking forms (WPForms or similar), Yoast SEO, image optimization/cache
- Performance: WebP images, lazy loading for high-res spa photos
